# Golden Intelligence Portfolio

An ultra-premium, storytelling portfolio website designed for a Data Scientist & Analytics Engineer.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **3D**: React Three Fiber (Three.js)

## Key Features
- **Golden Insight Reveal**: Signature interaction that reveals insights on hover.
- **Floating Executive Navbar**: Smart scroll-aware navigation.
- **Cinematic Hero**: Realistic 3D glass/gold element with mouse parallax.
- **Performance**: Lazy loaded 3D, optimized assets.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.
